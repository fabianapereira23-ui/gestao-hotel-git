Nome do Projeto: Sistema de Gestão de Reservas de Hotel
Autora:Fabiana Pereira

Descrição Geral: Este projeto consiste na criação de um sistema simples em Python para gerir reservas de um hotel. Este sistema permite gerir reservas, quartos e clientes de um cliente, aplicando práticas de controlo de versões com Git e GitHub. 

Funções principais do sistema: Registo e listagem de clientes; gestão de quartos (disponíveis/ocupados); criação e cancelamento de reservas.
